# FileBundlerCLI

כלי שורת פקודה (CLI) המיועד לאיחוד קבצי קוד למסמך טקסט אחד.
הכלי מאפשר סינון לפי שפת תכנות, מיון קבצים, הסרת שורות ריקות והוספת הערות מקור.

## מאפיינים:
- איחוד קבצים מרובים לקובץ אחד.
- סינון לפי סיומות קבצים.
- אפשרות למיון קבצים לפי שם או סיומת.
- אפשרות להסרת שורות ריקות מהקוד.
- יצירת קובץ תגובה (Response File) להרצה מהירה.

## טכנולוגיות:
- .NET / C#
- System.CommandLine
